# Talk.Ink

[Talk.ink](https://talk.ink) is a chat app with thread (Twist/Slack clone), the flagship demo of [Kontenbase](https://kontenbase.com) no-code backend API platform.

3 main features:

- Simple as email
- Realtime as chat
- Readable as thread
